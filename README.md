# 🐮 Moominsay

A cowsay with Moomins.

## 📅 My small weekend project

I like Moomins.

## 🏗️ Usage

### Characters

You can pass the `character` (`-c`) flag to pick the character. Currently available characters include:

- `moomin`
- `moominMama`
- `moominJump`
- `moominPappaSmall`

### Advice

You can pass the `type` (`-t`) flag to get `good` or `bad` advice.
